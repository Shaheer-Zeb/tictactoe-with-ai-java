## How to Run?

1. Clone the Repo
   
   ```git clone https://github.com/Shaheer-Zeb/tictactoe-with-ai-java.git```
2. cd into the "tictactoe-with-ai-java" directory.
3. Run the following command to compile your Java source code to .class (byte code).

   ```javac Main.java```
   
   Separate .class files would be created for each of the files (classes) i.e. Main, Game, and MinimaxAi.
4. Now finally, initiate the Java Runtime launcher with the following command
   
   ```java Main```
